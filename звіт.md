Тема: Розробка програмного засобу для візуалізації точок на координатній площині.
Мета: Написати програму, яка зчитує координати з текстового файлу, відображає їх на полотні розміром 960х540 пікселів та зберігає результат у графічному форматі.

Хід роботи:

Вибір інструментів: 
Для реалізації завдання було обрано мову програмування Python. Для роботи з графікою використано бібліотеку Matplotlib (модуль pyplot), оскільки вона має вбудовані методи для побудови графіків у декартовій системі координат та дозволяє точно контролювати розмір вихідного зображення (DPI та розмір у дюймах).

Алгоритм роботи програми:
Програма відкриває файл DS6.txt у режимі читання.
Здійснюється порядкове зчитування даних. Кожен рядок розбивається на дві частини (координата X та координата Y), які конвертуються у цілочисельний тип (int) та додаються до відповідних списків.
Створюється графічне вікно (figure) із розрахунковим розміром у дюймах, що відповідає 960x540 пікселів (при DPI=100).
Використано метод plt.scatter() для відображення точок, оскільки він найефективніший для побудови масиву окремих координат.
Встановлено межі осей (xlim, ylim) від 0 до ширини/висоти полотна відповідно, щоб початок координат (0;0) знаходився у лівому нижньому куті.
Результат збережено у файл result.png за допомогою методу savefig().

Результат: 
В результаті роботи програми отримано графічний файл result.png, на якому візуалізовано точки з заданого датасету відповідно до умов завдання.
